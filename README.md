# Aviation Operations Management System (AOMS)

## Project Overview
The Aviation Operations Management System (AOMS) is a simulated Agile project designed to demonstrate practical experience in Business Analysis and Project Management.  
The project focuses on managing flight operations, maintenance reporting, scheduling, and operational workflows using Azure DevOps.

## Tools Used
- Azure DevOps (Azure Boards)
- Agile Scrum Methodology
- GitHub (Portfolio Documentation)

## Key Responsibilities & Activities
- Created Epics, Issues, and Tasks using Azure DevOps Boards
- Organized work using Backlogs, Sprints, and Kanban Boards
- Performed sprint planning and task breakdown
- Tracked work progress across To Do, Doing, and Done stages
- Followed Agile best practices for requirement tracking and execution

## Project Structure
- Epics → Issues → Tasks hierarchy
- Sprint-based planning and execution
- Visual tracking using Azure Boards Kanban view

## Deliverables
- Azure DevOps Backlog
- Sprint Planning View
- Kanban Board Execution View

> This project is created for portfolio and learning purposes to demonstrate hands-on experience with Azure DevOps and Agile workflows.
